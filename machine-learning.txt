MINGGU 2

REGRESSION (nilai harus kontinu) : membangun model secara akurat memprediksi kasus yang tidak diketahui
- Variabel Independent : penjelas penyebab keadaan
- Variabel Dependent   : keadaan, target atau tujuan akhir yang coba diprediksi

TIPE MODEL REGRESSION
- Simple Regression	: ketika 1 var independen digunakan untuk memprediksi var dependen
 	 	  	  cth : prediksi C02emission vs EngineSize
	- Simple Linear Regression
		keunggulan
		- sangat cepat
		- no parameter tuning
		- mudah dipahami dan sangat dapat dijelaskan 	
	- Simple Non-linear Regression

- Multiple Regression	: lebih dari 1 var digunakan untuk memprediksi var dependen 
			  cth : prediksi C02emission vs EngineSize & Cylinders


RUMUS MSE (simple regression)
y(target) = teta0 + teta1.x1
Co2Emision = teta0 + teta1.EngineSize

Model Evaluasi 
- K-fold (setiap kelipatan mengunakan data test yang berbeda beda dari kelipatan lainnya)
  ++++ 1st
  ++++ 2nd
  ++++ 3rd
  ++++ 4th
  jika k = 4 kelipatan maka akan terbagi menjadi 4 bagian
  1st Fold = 25% test 75% train, akurasi = 80%
  2nd Fold = 25% test 75% train, akurasi = 84%
  3rd Fold = 25% test 75% train, akurasi = 82%
  4th Fold = 25% test 75% train, akurasi = 86%
  Accuracy = (80% + 84% + 82% + 86%)/4 = 83%
